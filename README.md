# Remote Opus - Vacatures

Een vacatureoverzicht in Indeed-stijl, volledig opgebouwd in de Remote Opus huisstijl
(kleuren, logo en fonts). Zoekbalk, filters, een vacaturelijst en een detailpaneel,
alles in een enkele statische pagina.

## Live demo

De live pagina staat op GitHub Pages en is direct te delen met klanten.

## Bestanden

- `vacatures.html` - de complete pagina met zoekbalk, filters, vacaturelijst en detailpaneel
- `index.html` - stuurt de bezoeker automatisch door naar `vacatures.html`
- `header.html` - het originele Remote Opus header voorbeeld waar de huisstijl uit komt

## Gebruik

Open `vacatures.html` in een browser, of bekijk de live demo. De vacatures staan als
voorbeelddata onderin het bestand en zijn eenvoudig aan te passen of te koppelen aan
een echte bron.
